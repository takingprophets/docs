## Overview

**2022 Model Pro+** is the ICTKESIK community's implementation of ICT's 2022 Unicorn Model — a structured entry framework that combines liquidity sweeps, displacement, Fair Value Gap delivery, and SMT divergence confirmation into a single graded setup tool. The indicator evaluates each potential setup against a configurable checklist of conditions and assigns it a letter grade (A+, A, B+, B, etc.). You can set a minimum grade to filter the display, keeping only the highest-conviction setups visible.

The indicator draws every layer of the Unicorn Model framework simultaneously: buyside and sellside liquidity levels, session liquidity references (NY AM, Asia, London, prior day and week highs/lows), macro time windows, HTF Fair Value Gaps, SMT divergence, ICTKESIK entry levels, limit levels (TP/BE/SL), breaker blocks, current timeframe FVGs, and market structure. A full-featured checklist and relationships table are included for live session workflow.

## Settings

### General

**Bias** — `Bullish` | `Bearish` | `Both`

Filters setups by directional bias. `Both` shows all valid setups regardless of direction.

---

**Model** — `Both` | `Silver Bullet` | `Unicorn`

Selects which ICT model variation to detect. `Unicorn` is the full 2022 model; `Silver Bullet` targets the 10–11 AM or 2–3 PM time-of-day setups specifically; `Both` runs detection for either.

---

**Require Macro Time** — toggle

When on, setups are only valid if they form within a recognized ICT macro window (10:10–10:40, 11:10–11:40, 2:10–2:40, 4:03–4:33 ET). This significantly tightens the signal filter.

---

**Remove Losing Setups** — toggle

When on, setups that did not result in a winning trade (price reached the SL before TP) are removed from the chart.

---

**Min Grade to Display** — grade selector (default: `B+`)

Filters out setups below this grade. Only setups with sufficient confluence are shown.

---

**ES Alignment** — toggle

When on, requires that the ES (S&P 500 futures) structural alignment supports the setup direction before displaying it.

---

## How We Use It

2022 Model Pro+ is the ICTKESIK community's all-in-one execution indicator. The grade filter is set to `B+` by default — this eliminates marginal setups and surfaces only those with strong liquidity, FVG, and structural confluence. The `Require Macro Time` toggle is a key filter during focused sessions: when enabled, only setups forming during the official ICT macro windows are shown, which dramatically improves precision. The macro box makes these windows immediately visible so the community knows exactly when to be watching for entries. The integrated limits (TP/BE/SL) mean there is no manual calculation needed — every element of the trade from entry to exit is drawn on the chart the moment the setup qualifies.
