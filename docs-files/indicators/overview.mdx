# Taking Prophets Indicator Suite

Our in-house suite covers every major concept in ICT (Inner Circle Trader) methodology. Each indicator is designed to work standalone or alongside the others as a complete charting system.

## Available Indicators

| Indicator | What It Does |
|-----------|-------------|
| FVG | Marks Fair Value Gaps on your chart |
| Order Blocks | Identifies institutional order blocks |
| Market Structure | Tracks highs, lows, and structure shifts |
| CISD | Change in State of Delivery |
| Breaker Blocks | Identifies failed order blocks that flip |
| Rejection Blocks | Marks candle wicks as rejection zones |
| BPR | Balanced Price Range detector |
| SMT Divergence | Tracks divergence between correlated pairs |
| ICT Killzones | Highlights high-probability session windows |
| ICT Liquidity Levels | Marks liquidity pools above/below price |
| HTF Candles | Displays higher timeframe candles inline |
| Prophet Model | The full Taking Prophets trade model |

<Tip>
  All indicators include alert functionality. Set alerts on your key levels so you never miss a setup.
</Tip>
